JPK ERP 사용 가이드

시스템 접속

  • PC: 브라우저에서 ERP 주소 접속
  • 모바일: 동일 주소 접속 시 자동으로 모바일 화면 전환
  • 빠른 검색: Ctrl+K → 차량번호 또는 고객명으로 즉시 검색

1. 통합 리포트

대시보드. 전체 운영 현황을 한눈에 파악합니다.

2. 입력

2-1. 운영업무

차량 운영에 필요한 모든 업무를 등록하는 핵심 페이지입니다.

화면 구성: 좌측 업무 카드 / 중앙 입력 폼 / 우측 이력·결과

업무설명
입출고센터정상출고·반납·강제회수·차량이동. 출발지/도착지 입력, 보험증권 업로드(출고 시), 운전자 연령 확인
차량케어센터정비·사고수리·상품화·세차 통합. 기본정보(일자/차번/작업구분/작업상태/출발지/도착지)는 고정, 작업구분에 따라 하단 세부 양식만 변경
고객센터통화/상담/컴플레인/문의 등 고객 응대 기록
사고접수사고 발생 시 접수. 사고형태(단독/쌍방), 가해/피해, 과실비율, 보험유형 선택
보험배서관리연령변경·갱신·신규·해지 등 보험 업무
과태료작업고지서 OCR 업로드 → 계약자 자동 매칭 → 고지서+확인서 PDF 생성 → ZIP 다운로드 또는 구글드라이브 저장

과태료작업 상세 흐름

  1. 좌측에서 "과태료작업" 선택
  2. 중앙: 고지서 스캔 파일(PDF/이미지) 드래그 업로드 → OCR 자동 처리 (진행율 표시)
  3. 우측: 매칭 결과 목록 — 매칭여부·차량번호·부과기관·위반일시·위반장소·금액·계약자·계약기간
  4. 행 클릭 → 새 탭에 미리보기 (고지서 1장 + 계약사실확인서 1장)
  5. 상단 버튼: 초기화 드라이브 저장 열기 처리완료 다운로드
  6. 처리완료 시 DB 저장 → 운영관리 과태료 내역에서 조회 가능
  7. 파일명: {부과기관} {차량번호} {YYMMDD_HHMM} {계약자} {차종}.pdf

차량번호 검색

모든 업무에서 차량번호 입력 시 회사코드·모델·계약자 자동완성 드롭다운이 표시됩니다.

2-2. 일괄 업로드

엑셀/CSV/PDF 파일을 업로드하여 대량 데이터를 등록합니다.

  • 자산 목록, 계약 목록, 통장 내역, 카드 내역 등
  • 보험증권 PDF → OCR 자동 파싱 (DB손해보험·렌터카공제조합 등 지원)
  • 과태료 고지서 PDF → OCR 자동 파싱

보험증권 OCR 특징

  • 디지털 PDF → 텍스트 직접 추출 (빠르고 100% 정확)
  • 스캔 이미지 PDF → Google Vision OCR
  • 차량번호 한글 인식 실패 시 → 숫자 매칭으로 자산 DB에서 자동 복원

2-3. 개별 입력

업무생성·자산등록·계약등록·고객등록 등 개별 데이터 직접 입력.

2-4. 입력 이력

최근 입력/수정 이력을 조회합니다.

회사·인사

공통 기능

차량번호 검색

모든 입력 폼에서 차량번호 입력 시 자동완성 드롭다운이 표시됩니다.

표시 형식: 차량번호 회사코드 모델 계약자

OCR (광학문자인식)

  • 디지털 PDF: 텍스트 레이어 직접 추출 (빠르고 정확)
  • 스캔 이미지/사진: Google Vision API OCR
  • 차량번호 패턴: 00가0000 또는 000가0000 자동 인식

구글드라이브 연동

과태료 확인서 PDF를 지정 드라이브 폴더에 자동 업로드합니다.

  • 드라이브 저장 버튼: PDF 생성 후 드라이브에 업로드
  • 열기 버튼: 드라이브 폴더 새 탭으로 열기
